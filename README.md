# java-lotto-precourse

### 🚀기능 구현

### 사용자로 부터 필요한 입력 받기
    - [✔️]  구입 금액 입력 받기   
      구입 금액이 나누어 떨어지지 않으면 예외처리   
    - [✔️]  당첨 번호 입력 받기   
      중복의 숫자가 존재하는 경우 예외처리   
    - [✔️]  보너스 번호 입력 받기
### 로또 로직 구현
    - [ ]  로또 구입 금액만큼 로또 발행 (오름차순 정렬)
    - [ ]  당첨 내역 계산
    - [ ]  수익률 계산
### 사용자에게 필요한 프롬프트 출력
    - [ ]  발행한 로또 수량 출력
    - [ ]  발행한 로또 번호 출력 (오름차순)
    - [ ]  당첨 내역 출력
    - [ ]  수익률 출력
